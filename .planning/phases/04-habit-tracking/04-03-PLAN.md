    - Daily streak: 5 consecutive days ending today returns 5
    - Daily streak: 5 consecutive days ending yesterday returns 5 (grace for today not yet done)
    - Daily streak: gap 2 days ago, completed yesterday and today returns 2
    - Daily streak: no completions returns 0
    - Daily streak: only today completed returns 1
    - Daily streak: with partial completion (count < targetCount) the day is NOT counted
    - Weekly streak: 3 consecutive weeks meeting target (e.g., 3 of 3 days per week) returns 3
    - Weekly streak: current week not yet met target, previous 2 weeks met returns 2
    - Custom streak: habit scheduled Mon/Wed/Fri, all scheduled days completed for 2 weeks returns streak of 6 scheduled days
    - Custom streak: non-scheduled days are ignored (completing on Tuesday doesn't count, missing Tuesday doesn't break streak)
    - bestStreak: scans all logs and finds the longest consecutive streak
    - completionRate: (days completed / total scheduled days) * 100 for a given date range
    - totalCompletions: sum of all log counts in a date range
  </behavior>
