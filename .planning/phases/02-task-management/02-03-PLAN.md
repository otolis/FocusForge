     - Default: null (no recurrence)
  2. When `weekly` selected: show 7 `FilterChip` widgets for Mon-Sun
     - Each chip toggles the day in `daysOfWeek` list
     - Use ISO weekday (1=Mon, 7=Sun)
     - Default: none selected (user must pick at least one)
  3. When `monthly` selected: show a `DropdownButtonFormField<int>` for day of month (1-31)
     - Default: current day of month
  4. When `custom` selected: show a `TextFormField` for "Every N days"
     - Input type: number, validator: must be > 0
     - Default: 2
  5. When `daily` selected: no additional options needed
- Call `onChanged` with null when "None" selected, or with populated `RecurrenceConfig` otherwise
