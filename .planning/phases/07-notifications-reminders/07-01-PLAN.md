**1. Create `lib/core/services/notification_service.dart`:**

Singleton pattern (like research example). This is the central notification orchestrator on the Flutter client.

```dart
class NotificationService {
  static final NotificationService _instance = NotificationService._();
  factory NotificationService() => _instance;
  NotificationService._();

  final FlutterLocalNotificationsPlugin _localNotifications = FlutterLocalNotificationsPlugin();
```

**Android Notification Channels (3 channels):**
- `task_reminders` -- "Task Reminders" -- Importance.high (heads-up display)
- `habit_reminders` -- "Habit Reminders" -- Importance.defaultImportance
- `planner_notifications` -- "Planner Notifications" -- Importance.defaultImportance

**`Future<void> initialize()` method:**
1. Call `FirebaseMessaging.instance.requestPermission()` for Android 13+ POST_NOTIFICATIONS
2. Get the Android plugin via `_localNotifications.resolvePlatformSpecificImplementation<AndroidFlutterLocalNotificationsPlugin>()`
3. Create all 3 notification channels via `androidPlugin?.createNotificationChannel(channel)`
4. Initialize `_localNotifications` with `InitializationSettings(android: AndroidInitializationSettings('@mipmap/ic_launcher'))`, `onDidReceiveNotificationResponse: _onNotificationTap`, `onDidReceiveBackgroundNotificationResponse: onBackgroundNotificationAction`
5. Listen for foreground messages: `FirebaseMessaging.onMessage.listen(_showLocalNotification)`
6. Set up background message handler: `FirebaseMessaging.onBackgroundMessage(firebaseMessagingBackgroundHandler)`
7. Set up `FirebaseMessaging.onMessageOpenedApp.listen` for deep link when app opens from background notification tap
8. Check `FirebaseMessaging.instance.getInitialMessage()` for app opened from terminated state

**`_channelForType(String type)` helper:**
Returns the appropriate `AndroidNotificationChannel` based on reminder_type string:
- `'task_deadline'` -> task_reminders channel
- `'habit_reminder'` -> habit_reminders channel
- `'planner_summary'` or `'planner_block'` -> planner_notifications channel

**`void _showLocalNotification(RemoteMessage message)` method:**
Extract `data` map from message. Get channel via `_channelForType(data['type'])`. Call `_localNotifications.show()` with:
- `id`: `message.hashCode`
- `title`: `data['title'] ?? ''`
- `body`: `data['body'] ?? ''`
- `NotificationDetails` with `AndroidNotificationDetails` using the channel, `subText: data['insight']` for adaptive timing transparency, and two `AndroidNotificationAction`s: `('complete', 'Complete', cancelNotification: true)` and `('snooze', 'Snooze', cancelNotification: true)`
- `payload`: JSON-encode `{'type': data['type'], 'item_id': data['item_id'], 'route': data['route']}` for both tap and action handling

**`_onNotificationTap(NotificationResponse response)` method:**
Parse payload JSON. Extract `route` field. Navigate via `GoRouter` using a global navigator key: `GoRouter.of(navigatorKey.currentContext!).push(route)`. Import navigatorKey from app_router or use a separate global key.

**FCM Token Lifecycle -- `Future<void> manageFcmToken(String userId, NotificationRepository repo)` method:**
1. Get token: `final token = await FirebaseMessaging.instance.getToken()`
2. If token not null, call `repo.storeFcmToken(userId, token)`
3. Listen for refresh: `FirebaseMessaging.instance.onTokenRefresh.listen((newToken) => repo.storeFcmToken(userId, newToken))`

**`Future<void> clearToken(String userId, NotificationRepository repo)` method:**
Call `repo.clearFcmToken(userId)`. Call `FirebaseMessaging.instance.deleteToken()`.

**Top-level background handler (MUST be top-level, not inside class):**

```dart
@pragma('vm:entry-point')
Future<void> firebaseMessagingBackgroundHandler(RemoteMessage message) async {
  await Firebase.initializeApp();
  final plugin = FlutterLocalNotificationsPlugin();
  await plugin.initialize(
    const InitializationSettings(
      android: AndroidInitializationSettings('@mipmap/ic_launcher'),
    ),
    onDidReceiveBackgroundNotificationResponse: onBackgroundNotificationAction,
  );
  // Display notification with action buttons (same logic as foreground)
  final data = message.data;
  final channelId = _channelIdForType(data['type'] ?? '');
  await plugin.show(
    message.hashCode,
    data['title'] ?? '',
    data['body'] ?? '',
    NotificationDetails(
      android: AndroidNotificationDetails(
        channelId, _channelNameForId(channelId),
        importance: channelId == 'task_reminders' ? Importance.high : Importance.defaultImportance,
        subText: data['insight'],
        actions: [
          const AndroidNotificationAction('complete', 'Complete', cancelNotification: true),
          const AndroidNotificationAction('snooze', 'Snooze', cancelNotification: true),
        ],
      ),
    ),
    payload: jsonEncode({'type': data['type'], 'item_id': data['item_id'], 'route': data['route']}),
  );
}
```
